# Bookstore Management Application

A **Spring Boot** project demonstrating role-based access control, CRUD operations, and user authentication for a simple bookstore system.

## Features

- **User Roles**:
  - **ADMIN**: Can create, read, update, and delete (CRUD) books and manage users.
  - **USER**: Can view book information only.
  - Admin user auto-created at startup (`admin@gmail.com / admin123`)  


  ## Tech Stack

- HTML,CSS,Spring Boot
- Spring Boot  
- Spring Data JPA  
- Spring Security  
- Thymeleaf (for views)  
- MySQL
